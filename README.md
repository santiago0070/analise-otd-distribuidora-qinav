Projeto Supply Chain - Distribuidora Qinav 🚚

Este projeto apresenta uma análise completa de OTD (On-Time Delivery) e faturamento para uma distribuidora de produtos hospitalares e médicos. O foco principal é identificar gargalos logísticos e o impacto financeiro de atrasos nas entregas.

🎨 Branding e Identidade Fictícia

Empresa Fictícia: A Qinav Distribuidora é uma marca simulada criada integralmente por mim para este estudo de caso. 

Criação de Logo e Nome: Desenvolvi o nome e o logotipo para conferir realismo ao projeto e demonstrar habilidades em design de interface e storytelling de dados.

Layout do Dashboard: O plano de fundo e a estrutura visual foram desenhados para refletir um ambiente corporativo profissional.

📊 Principais Insights do Projeto
Nível de Serviço (OTD): Atualmente em 60%, indicando uma necessidade crítica de otimização logística.

Gargalo de Entrega: O produto Suturas apresenta o maior tempo médio de entrega (3 dias) e o menor índice de pontualidade (41%).

Concentração de Receita: Os produtos OPME e Endoscópios concentram o maior faturamento, totalizando cerca de R$ 274 Mi.

Impacto Financeiro: Aproximadamente R$ 80 Mi da receita total (29%) estão vinculados a pedidos com entrega em atraso.

🛠️ Tecnologias e Metodologias
Python (Pandas & NumPy): Utilizados para a geração da base de dados sintética, tratamento de dados e cálculo de indicadores.

Power BI: Desenvolvimento de dashboards interativos com métricas de rentabilidade, giro de estoque e eficiência operacional.

Modelagem de Dados: Criação de colunas calculadas e medidas em DAX para suporte à tomada de decisão executiva.
